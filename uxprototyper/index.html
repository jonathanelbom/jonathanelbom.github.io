transformMe

var array1 = [3,1,4,5,2];
function transformMe( a ) {
// sort a ascending numerically
a = a.sort();
// return a, a reversed, and a. Array.prototype.reverse reverses in place (while sort and concat return a new array). make sure the reverse call is operating on a concatenated copy
return a.concat( a.concat().reverse(), a );
}
array1 = transformMe(array1)
console.log('array1:',array1);
// outputs array1: [1,2,3,4,5,5,4,3,2,1,1,2,3,4,5]

Protoype UX notes:

Since this is just a demonstration and was done on a shorter timeline (i got this exercise on fri 6/10), not all of these UX features are fully fleshed-out. They are there to show knowledge and consideration of good UX design principles and implementatin strategies.

  • Screen elements adjust to window resize (e.g. scroll panels adjust to use fully designate heights, modal size adjust)
  • Scroll overflow shadow hints - independent shadow hints on top and bottom (if not bottom of page) of scroll panels to indicate more content
  • Grid system
  • Partially-responsive design. Container resizes for screen width, but grid columns do not collapse to 1 column at small width. Use of flexbox CSS allows for smart resizing of dynamic content layout
  • Body base font size and use of em allow for easy global relative adjustment of font size (use the font-size widget in the secondary header to try this out)
